# Assets

Pasta reservada para materiais visuais do projeto.

## Organizacao sugerida

- `brand/`: logos, paleta, identidade do cliente
- `img/`: fotos, texturas e imagens de apoio
- `icons/`: icones e elementos graficos

## Objetivo

Centralizar tudo que reforca a percepcao de valor da aplicacao:
visual forte, acabamento profissional e coerencia com o universo do agro.
